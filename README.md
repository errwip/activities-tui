# activities-tui

This repo is a Terminal User Interface for my other project. [CSV DataBase](https://github.com/errwip/toy-csvdb) 
I'm making the TUI using [ratatui](https://github.com/ratatui/ratatui) as dependancy.

For now the TUI autoamtically reads the entire test db and displays it. Nothing more.
